# 💪 FitTrack - Fitness Tracker

An interactive web application to log daily exercises, track workouts, and monitor fitness progression.

---

## 📌 Overview

**Fitness-Tracker** is a frontend application aimed at gym-goers. It allows users to record workout types, durations, and sets, dynamically updating list metrics and tracking performance over time.

---

## ⚙️ Tech Stack

| Technology | Purpose |
| ---------- | ------- |
| HTML5      | UI layout |
| CSS3       | Dark-themed modern layout |
| JavaScript | Log manipulation and localStorage logic |

---

## 🚀 Key Features

* 🏋️ **Workout Logger:** Add customized workouts (sets, reps, time).
* 💾 **Persistence:** Automatically saves user workouts locally to `localStorage`.
* 📊 **Progress Summary:** Shows dynamic summaries of active hours and exercise counts.
* 📱 **Mobile UI:** Easily accessible layout on mobile phones.

---

## 🏗️ Project Structure

* `index.html` - Main interface page.
* `style.css` - Custom styles and dashboard panels.
* `main.js` - Logic controlling storage and dashboard rendering.
